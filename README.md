# Node_Template_Engine

This is a Functional Team Template Generator for a team consiting of a Manager, Engineer, and Intern. 

Deployed Link: https://ckalama.github.io/Node_Template_Engine/

Programming Tools: HTML, CSS, Bootstrap, Javascript, Node.js

Description: Built strictly through node.js. It requires usage of the Terminal to generate a Template created through Javascript and node.js. You will be prompted to answer a series of questions that will designate what position you are. From there, you will be prompted to add more Employees or not and generate a functioning HTML page consisting of your teammates. 

Installation: You will need to have node.js installed in your Terminal. As well as npm Packages: inqurier, fs, and jest.js for testing purposes

Usage: User accesses the Terminal, inserts "node app.js" from there, you will be prompted basic questions on your Employees. Based on your answers you will generate infinite employees or generate a HTML page with your inserted Employees.
